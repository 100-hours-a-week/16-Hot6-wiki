린캔버스

문제

  • 현재 내 데스크 환경에 맞는 레퍼런스를 찾기 어려워 구매 결정을 못함
  • 레퍼런스를 찾더라도 어디서 아이템을 구매해야 할지 모름
  • 추천된 아이템이 들어있거나 가성비 좋은 제품이면 더 좋겠음

솔루션

  • AI가 현재 내 데스크 환경에 어울리는 제품 이미지를 생성하여 추천해줌
  • 추천된 아이템의 실제 상품을 구매할 수 있도록 URL과 함께 제공
  • 특가 제공 및 커머스 링크 연동

고유 가치 제안

  • 실제 내 데스크에 합성해주므로 레퍼런스 방면에서 굉장히 효과적
  • AI가 생성한 이미지 기반으로 실제 상품을 추천하고 바로 구매 가능
  • 이미지 중심의 개인 맞춤형 추천으로 사용자가 상상하던 이상적인 쇼핑 경험을 제공

경쟁 우위

  • 실제 이미지로 레퍼런스를 생성하여 추천
  • 추천된 아이템에 실제 상품 링크 연동
  • 사용자 환경을 이해하고 시각적으로 제안하는 차별화된 AI 기술

고객 세그먼트

  • 데스크를 꾸미고 싶지만 어떻게 해야 할지 모르는 사람
  • 아이템을 구매하고 싶지만 실제로 잘 어울리는지 확인하고 싶은 사람
  • 추천된 상품을 저렴하게 사고자 하는 사람

기존 대안

  • 커뮤니티에서 'OO 데스크 추천' 같은 게시글 찾아봄
  • 유사 데스크셋업을 검색하여 레퍼런스를 수동으로 수집
  • 원하는 아이템을 검색을 통해 개별 구매

핵심 지표

  • 일간 사용자 수
  • 추천 서비스 통한 상품 클릭률
  • 추천 서비스 후 실제 구매 전환율

상위 개념

  • text to image 방식이 아닌 image to image 방식으로 사용이 더 직관적이고 편리함
  • 추천된 아이템 바로 구매 가능
  • 우리 서비스에서 특가 진행

채널

  • 검색을 통한 유입
  • Instagram 등 SNS에서 추천 → 실사용 후기 형식의 큰 반응
  • 네이버 블로그 리뷰 체험단 혹은 유튜버 협업 등 인플루언서 활용 홍보

얼리어답터

  • 데스크 셋업에 관심이 많고 구매 전에 공간에 어울릴지 미리 확인하고 싶은 사용자
  • 시각적 정보에 민감하고 감성적인 데스크 환경을 꾸미는데 취향이 뚜렷한 소비자
  • 트렌드에 민감하고 SNS에 자신의 데스크 셋업을 자주 공유하는 사용자

비용 구조

  • 고정비: 인건비 0원, 기타 고정비 240만원
  • 변동비: AWS 100만원 지원, GCP 약 $1,800 무료, DALL-E 3 API 월 15만원, 특가 비용 월 1만원
  • 총계: 월 136만원, 인당 약 23만원

수익 구조

  • 사업자 등록을 할 수 없어 수익창출 불가
  • 커머스와 계약 체결 후 커미션
  • 광고 수익 (사이트 배너 등)
